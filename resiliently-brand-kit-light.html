Rules
Dual-Mode Rules
01
Two backgrounds, one system
Dark mode (#000) for video canvas. Light mode (#fff) for articles, blog posts, documents. Same fonts, same hierarchy, same handwritten accent. Only the color values shift.
02
Accent darkens on white
Dark: #5ec1ff. Light: #0077b6. Same hue, different lightness. All semantic colors (green, red, gold) darken proportionally for WCAG contrast.
03
Same 8 production rules
Bebas Neue caps, Caveat lowercase, blue accent, black canvas (or white), four fonts only, no rounded corners, German-first, no company names. The dark-mode rules apply identically to light mode.
04
Light mode = editorial pages
Use light mode for blog posts, newsletter articles, PDF reports, and any reading-focused context. Dark mode is for video and motion graphics only.
