Production Rules
Non-Negotiable
01
Bebas Neue always in caps
Every headline, every stat, every label set in Bebas Neue is uppercase. No exceptions. Tall, condensed, brutal. The caps are the authority.
02
Caveat always lowercase
Handwritten annotations are always lowercase. The contrast between ALL CAPS machine and lowercase human IS the brand.
03
Blue is the only accent
#5ec1ff for highlights, stats, handwritten annotations, CTAs. Green/red/gold are semantic (coverage/exclusion/premium). Never decorative.
04
Black canvas, never flat
#000000 is the background. Add depth with #0a0a0a cards and #1a1a1a surfaces. Never use a colored background.
05
Four fonts, four roles
Bebas Neue = headlines. Caveat = handwritten. DM Sans = body. JetBrains Mono = data. Never cross roles. Never add a fifth font.
06
No rounded corners
Sharp edges everywhere. Cards, tags, buttons, PiP. 0px radius. Editorial sharpness, not startup softness.
07
German-first on-screen text
All motion graphics text in German. English only for Pillar B code demos.
08
No company names
Never show Zurich, AXA, Allianz, or real client names. Fictitious scenarios only.
